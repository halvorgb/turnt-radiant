## turnt-radiant
radiant configuration for turnt

## Contents:
- Greatly simplified entity file of what is currently supported
- Dev textures (optional install)
- Build menu
- Some common shaders for radiant internals (clip is translucent etc.) (turnt does not read shaders.)

## Installation
Download zip file from releases, extract into radiant folder.
